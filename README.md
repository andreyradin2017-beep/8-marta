# 8 Марта - Поздравительный Landing Page

Репозиторий содержит легковесную веб-страницу (HTML/CSS/JS), разработанную для создания интерактивного поздравления с 8 Марта (от маркетинговой группы "Текарт").

## Особенности проекта

- **Фоновое видео** с автовоспроизведением и возможностью включения/выключения звука.
- **Эффект параллакса (Parallax)** элементов дизайна при движении указателя мыши.
- **Адаптивный дизайн (Responsive)** — идеальное отображение на компьютерах, планшетах и мобильных телефонах. Используются функции `clamp()` для плавного масштабирования текста и элементов.
- **Сверхвысокая производительность** — страница использует чистый (Vanilla) JavaScript и СSS, не требуя сборщиков или тяжелых библиотек (React, Vue, jQuery и др.).
- **Современная типографика** — интеграция web-шрифтов Onest (из Google Fonts).

## Структура репозитория

- `index.html` — Входная точка проекта.
- `css/style.css` — Стилизация и адаптивность, CSS-переменные, анимации.
- `js/main.js` — Логика интерактивного взаимодействия (управление звуком, параллакс-скроллинг элементов).
- `video.mp4` — Файл фонового видео-ролика.
- `images/` — Иконки в формате векторной графики SVG.
- `docs/SPEC.md` — Подробная техническая спецификация проекта.

## Инструкция по запуску

Так как проект написан с использованием базовых Web-технологий (HTML5, CSS3, ES6), для его запуска не требуется установка Node.js или дополнительных зависимостей с помощью npm.

### Способ 1: Прямой запуск

Просто откройте файл `index.html` в любом современном браузере.
_Примечание: в локальном режиме без веб-сервера браузеры могут блокировать автозапуск видео со звуком в зависимости от своих внутренних политик безопасности._

### Способ 2: Запуск с помощью локального сервера (Рекомендуется)

Для тестирования всех возможностей (особенно при разработке расширенного функционала, затрагивающего CORS) рекомендуется использовать простейший Live-сервер:

- В среде VS Code: установите расширение **Live Server** и нажмите "Go Live".
- Через терминал с Python (если установлен): `python -m http.server 8000`. После чего откройте `http://localhost:8000`.
- Через Node.js (npx): `npx serve .`

## Дальнейшая разработка (Contribution)

Все исходные стили находятся в файле `css/style.css`.

- Конфигурация Z-индексов, цветовой схемы и параметров наложения видеофона может быть изменена модификацией переменных в секции `:root`.
- Для изменения скорости или направления параллакс-эффекта декоративных элементов (например, SVG-цветов) достаточно изменить значение data-атрибута `data-speed` прямо в HTML-разметке:

```html
<div class="parallax-layer js-parallax" data-speed="-0.15">
  <img src="..." alt="..." />
</div>
```

## Требования

Все графические UI-элементы настоятельно рекомендуется импортировать в векторном формате `.svg`. Видео использовать в формате `.mp4` с оптимальной компрессией (к примеру, H.264), отключенными субтитрами.
